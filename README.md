## ✨ Features

- **Modern UI/UX Architecture**: Features a seamless blend of Glassmorphism and Brutalism aesthetics. Uses high-contrast UI components, subtle scroll animations, and dynamic typewriter effects for a premium feel.
- **Fully Responsive**: Carefully crafted CSS ensures perfect display on mobile devices, tablets, and large desktop monitors.
- **Dynamic Content**: Displays detailed project cards complete with categorizations, descriptions, GitHub links, and live deployment links.
- **Live Contact Form**: A fully functional, secure contact form powered by **Vercel Serverless Functions** and **MongoDB**. Visitors can send messages directly to a cloud database without needing an active backend server.
- **Lightning Fast Performance**: Built using Vite and React 19 for instantaneous Hot Module Replacement (HMR) during development and highly optimized static builds for production.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React.js (v19)
- **Language**: TypeScript
- **Bundler/Tooling**: Vite
- **Icons**: `lucide-react` & `react-icons`
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid, Custom Animations)
- **Typography**: Google Fonts (Outfit & Inter)

### Backend & Database
- **API**: Vercel Serverless Functions (Node.js)
- **Database**: MongoDB Atlas (accessed via Mongoose)

### Deployment
- **Hosting**: Vercel (with continuous CI/CD integration from GitHub)
